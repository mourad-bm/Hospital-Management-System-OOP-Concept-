# Hospital-Management-System-OOP-Concept-
Hospital Management System is an OOP-based project for LifeCare Hospital. It manages doctors, nurses, receptionists, patients, and admins with role-based access. Each user can only perform assigned tasks, ensuring security, organization, and clear separation of responsibilities in hospital operations.
🏥 Hospital Management System (OOP)
📌 Description

This project is a simple Hospital Management System built using Python . It manages different roles such as doctors, nurses, receptionists, patients, and admins with role-based access.

🎯 Features 

** Patient registration
** Appointment management
** Medical record tracking
** Role-based permissions
🧱 OOP Concepts

Encapsulation
Inheritance
Polymorphism

🛠️ Technology ( environement)

Python (PyCharm) python version 3.11.9

🚀 How to Run
python main.py
