# 🏥 Hospital Management System (OOP)

Hospital Management System is an OOP-based project for LifeCare Hospital. It manages doctors, nurses, receptionists, patients, and admins with role-based access. Each user can only perform assigned tasks, ensuring security, organization, and clear separation of responsibilities in hospital operations.

## 📌 Description

This project is a simple Hospital Management System built using Python. It manages different roles such as doctors, nurses, receptionists, patients, and admins with role-based access control.

## 🎯 Features

- Patient registration
- Appointment management
- Medical record tracking
- Role-based permissions

## 🧱 OOP Concepts

- Encapsulation
- Inheritance
- Polymorphism

## 🛠️ Technology

- Python
- PyCharm
- Python version: 3.11.9
